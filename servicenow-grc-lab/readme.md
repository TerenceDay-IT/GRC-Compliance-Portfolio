# ServiceNow GRC Compliance Implementation Lab

## Objective
Designed and implemented an IT Governance, Risk, and Compliance (GRC) framework within ServiceNow to monitor and enforce organizational security controls, achieving a 100% compliance rating for critical IT infrastructure entities.

## Environments & Tools Used
* ServiceNow (Australia Release) - Policy and Compliance Management Application
* GRC Workbench

## Project Milestones & Architecture
1. **Entity Configuration:** Defined and mapped critical IT infrastructure assets (`Splash Log Ent - IT infrastructure`) to track compliance posture.
2. **Control Objective Template Design:** Formulated master compliance templates, including *Identity and Access Management (IAM) Policy* and *Corporate Password Complexity Standards*.
3. **Control Implementation:** Generated and deployed granular, actionable controls targeting multi-factor authentication (MFA), 14-character minimum password constraints, and 24-hour offboarding access termination logs.
4. **Attestation & Assessment Workflow:** Configured GRC Attestation questionnaires, attached evidence logs, and moved records from Draft through Attestation to Monitor states to achieve a verified Compliant status.
